O Banco de Projetos

O Banco de Projetos de Pesquisa é um módulo do Sistema de Gestão Acadêmica – SIGA/UFPR. Este novo sistema importa os dados de projetos de pesquisa, juntamente com dados da equipe participante e de produção intelectual vinculada ao referido projeto, diretamente do currículo Lattes de cada pesquisador.

Dessa forma, o registro de um novo projeto de pesquisa ou alteração dos dados de um projeto existente é feito diretamente por meio de atualização do currículo lattes do pesquisador, sendo então refletido na base de dados da UFPR, mediante opção “Sincronizar com o Lattes”.

Cabe ressaltar que o docente deverá fazer a vinculação da produção científica e das orientações diretamente no currículo lattes.
